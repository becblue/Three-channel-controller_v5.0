# 三通道控制器 v5.0 全新实现项目

日期：2026-06-18

这个目录是三通道控制器固件 v5.0 的全新实现项目。

旧版 STM32/Keil 工程保留在 `Three-channel controller_v3.0` 中，只作为硬件资料、旧驱动素材和行为参考。v5.0 不继承旧工程架构，不直接沿用旧主循环、旧中断逻辑、旧看门狗逻辑和旧报警路径。

## 项目规则

- 架构文档放在 `docs/`。
- 新固件头文件放在 `Core/Inc/`。
- 新固件源文件放在 `Core/Src/`。
- 不向旧工程目录添加 v5.0 实现文件。
- 不在旧工程中混入 v5.0 新架构代码。
- 所有 `.md` 文档使用中文。
- 所有代码注释使用中文。
- 所有新增文件统一使用 UTF-8 编码。

## 实现目标

从已冻结的全新规划出发，建立边界清晰、单向依赖、方便验证的固件结构。

目标模块包括：

- 板级 IO 层
- 输入管理
- 通道控制
- 继电器管理
- 反馈监控
- 安全报警管理
- 外部电源监控
- 温度管理
- 报警输出
- 显示管理
- 看门狗管理
- 维护服务
